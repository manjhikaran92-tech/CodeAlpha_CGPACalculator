# CGPA Calculator

A C++ program to calculate CGPA based on courses, grades, and credit hours.

## Features
- Input number of courses
- Enter grade points and credit hours for each course
- Displays individual course details
- Calculates and displays final CGPA

## How to Run
1. Clone this repository
2. Open terminal and navigate to folder
3. Compile: g++ cgpa.cpp -o cgpa
4. Run: ./cgpa

## Technologies Used
- C++
- Standard Template Library (STL)
